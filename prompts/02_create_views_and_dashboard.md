# Prompt 02: Create Views and Dashboard

**Context:** You are a Notion UX/UI Designer.
**Goal:** Create professional views and a central dashboard for the Distribution Radar.

**Instructions:**
1. Access the following databases:
   - "Mapa de Distribuição Mansão Maromba"
   - "Alvos de Prospecção"
2. Create the following views if they don't exist:
   - **Prospecção Funnel**: A Board view of "Alvos de Prospecção" grouped by `Funnel Stage`.
   - **Distribution Map**: A Map view of "Mapa de Distribuição" using the `Endereço` property.
   - **Performance Gallery**: A Gallery view of active points showing their `Volume Mensal`.
3. If the MCP client supports page creation, draft a "Main Radar Dashboard" page that links to these databases and provides a high-level summary of:
   - Total active points.
   - Total leads in the funnel.
   - Top performing regions.
4. Ensure the naming is consistent and professional.
